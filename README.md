## Compatibility

passwd\_table 在 Python 2.7 下测试通过。

## Installation

克隆 passwd\_table

    git clone https://github.com/KleeTaurus/passwd_table.git

创建并激活虚拟环境

    cd passwd\_table
    virtualenv venv
    source venv/bin/activate

安装依赖包

    pip install -r requirements.txt

## Usage

在命令行输出随机密码表

    python passwd_table.py
